# WorkBridge Project

A platform connecting students with authentic and flexible part-time jobs.

## Tech Stack
- **Backend**: .NET 8 Web API
- **Frontend**: React (Vite)
- **Database**: SQL Server (Planned)

## Getting Started

### Backend
1. Navigate to `Backend`.
2. Run `dotnet run --project WorkBridge.API`.

### Frontend
1. Navigate to `Frontend`.
2. Run `npm install` (if not already done).
3. Run `npm run dev`.
